# 命理知识库说明

问元知识库用于**规则层摘要与 AI 锚定**，不替代程序排盘。

部分结构化摘要参考了开源项目 [bazi-skill](https://github.com/jinchenma94/bazi-skill)（MIT License）中的典籍整理思路，并已按问元「子平综参、不作单一派系独断」原则改写为可检索片段。

## v1.3 架构

```
排盘 (bazi.py)
  → 子平综参 (mingli.py)
      ├── 滴天髓：旺衰、体用、从化倾向
      ├── 穷通宝鉴：120 组调候
      ├── 子平真诠/三命通会：格局、清纯度
      ├── 渊海子平：喜用倾向（扶抑/调候/通关）
      └── 三命通会：常用神煞（辅助）
  → 典籍检索 (knowledge.py + snippets.py)
  → AI 锚定 (ai.py，要求标注出处)
```
