# 从革格—从旺命（命例06）

## 八字

| | 年柱 | 月柱 | 日柱 | 时柱 |
|--|------|------|------|------|
| 天干 | 戊 | 辛 | 庚 | 丁 |
| 地支 | 申 | 酉 | 申 | 亥 |

**日主：** 庚金（阳金）
**五行分布：** 金旺极（三申一酉）、土生金、水有根、火无根

## 大运

| 大运 | 年龄 |
|------|------|
| 壬戌 | 2-11 |
| 癸亥 | 12-21 |
| 甲子 | 22-31 |
| 乙丑 | 32-41 |
| 丙寅 | 42-51 |
| 丁卯 | 52-61 |

## 任铁樵原评（主旨）

庚金生于酉月，得令；年支申、日支申 — 三申一酉，金旺极；年干戊土偏印生金；时干丁火官星虚浮无根。

任氏评曰：**金旺极矣，不可逆也。丁火虚浮，不能克金，反为旺金所侮。此命从金之旺势，所谓从革格也。**

## 解读

### 格局分析
- 庚金生于酉月 — 帝旺之地
- 年柱戊申 — 土生金
- 日柱庚申 — 金坐禄
- 时支亥水 — 金生水泄秀
- 时干丁火 — 官星虚浮无根

此为**从革格**（金气专旺）。从革格的条件：
1. 庚辛金生于秋月（申酉戌）得令
2. 命局金旺极（三申一酉）
3. 地支无根火来破格
4. 即使有一点火，也虚浮无根（如本例丁火）

### 用神
- **喜神**：金（比劫）、土（印）
- **喜神（次）**：水（食伤泄秀）
- **忌神**：火（官杀逆势）

### 从格的命理原则
1. **旺极不可逆**: 从格之命，逆势则灾
2. **顺泄为佳**: 金旺喜水泄（如本例时支亥水）
3. **最忌破格**: 若大运走火地，丁火得根攻金 — 灾祸立至

### 大运应期
- **壬戌、癸亥运**：水运泄秀 — 才华出众，少年得志
- **甲子运**：甲木财星虚浮，子水泄金 — 平稳
- **乙丑运**：乙木虚浮，丑土生金 — 尚可
- **丙寅、丁卯运**：**最凶** — 火运破格，逆金之势，灾祸频生

### 命理启示
此命例说明：**顺局从旺之命，最怕逆势大运。** 从格之人在顺运中顺风顺水，但在逆运中比常人更难承受打击。
