# 伤官佩印—文贵命（命例09）

## 八字

| | 年柱 | 月柱 | 日柱 | 时柱 |
|--|------|------|------|------|
| 天干 | 甲 | 丁 | 癸 | 丙 |
| 地支 | 子 | 卯 | 巳 | 午 |

**日主：** 癸水（阴水）
**五行分布：** 木火旺极、水有微根、金无

## 大运

| 大运 | 年龄 |
|------|------|
| 戊辰 | 8-17 |
| 己巳 | 18-27 |
| 庚午 | 28-37 |
| 辛未 | 38-47 |
| 壬申 | 48-57 |
| 癸酉 | 58-67 |

## 任铁樵原评（主旨）

癸水生于卯月，食神当令；月干丁火偏财透出，时干丙火正财，年干甲木伤官高透。癸水仅年支子水为根，身弱而伤官财星两旺。

任氏评曰：**伤官佩印，文贵之格。此命妙在日支巳中戊土正官暗藏，伤官见官而有印化之，反成贵格。**

## 解读

### 格局分析
- 癸水生于卯月 — 食神当令
- 年干甲木伤官 — 高透有力
- 月干丁火偏财、时干丙火正财 — 财星通明
- 年支子水 — 日主唯一根气
- 日支巳火 — 财旺之地，暗藏戊土正官

此为 **伤官格**。伤官透出、财旺身弱，关键在于：

**伤官佩印的奥义：** 伤官格见印星（金）制伤官，是为"伤官佩印"，主文贵。此命不见金印，但日支巳中戊土正官与甲木伤官形成"伤官见官"，却又以巳中庚金为暗印（庚生癸水），暗藏玄机。

### 用神
- **喜神**：金（印星制伤官生身）、水（比劫帮身）
- **忌神**：木（伤官泄身）、火（财星耗身）

### 大运应期
- **戊辰运**：戊土正官透出 — 伤官见官，少年波折
- **己巳运**：财官旺地 — 身弱不胜财官，劳碌
- **庚午运**：**庚金正印透出** — 伤官佩印成格，文贵
- **辛未运**：辛金偏印帮身 — 学识精进
- **壬申运**：水金运帮身 — 富贵可期
