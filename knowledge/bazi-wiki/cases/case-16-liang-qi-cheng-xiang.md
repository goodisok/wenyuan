# 两气成象—金木相战（命例16）

## 八字

| | 年柱 | 月柱 | 日柱 | 时柱 |
|--|------|------|------|------|
| 天干 | 庚 | 乙 | 庚 | 丙 |
| 地支 | 寅 | 酉 | 申 | 子 |

**日主：** 庚金（阳金）
**五行分布：** 金旺、木旺、水有根、火微

## 大运

| 大运 | 年龄 |
|------|------|
| 丙戌 | 7-16 |
| 丁亥 | 17-26 |
| 戊子 | 27-36 |
| 己丑 | 37-46 |

## 任铁樵原评（主旨）

庚金生于酉月，得令；年支寅木偏财、月干乙木正财；日支申金、年干庚金比肩帮身。

任氏评曰：**金木相战，两气成象。此命金木各占半壁江山，关键在于有水通关。申子半合水局，子水食神化金生木，使金木之战转为金生水、水生木的流通之势。通关系全局之枢纽。**

## 解读

### 格局分析
- 庚金生于酉月 — 帝旺
- 年柱庚寅 — 比肩坐财
- 月柱乙酉 — 正财坐劫财
- 日柱庚申 — 比肩坐禄
- 时柱丙子 — 七杀坐食神

**两气成象（金木相战）**：

此命的要害在于金与木的对峙：
- **金**：庚庚比肩 + 申酉金旺地
- **木**：寅木 + 乙木透出

如果无水，金木相战则两败俱伤。幸有 **时支子水通关**：
```
金（旺）→ 生 → 水（子）→ 生 → 木（寅乙）
```

### 用神
- **喜神**：水（食伤通关）、火（官杀制金卫木）
- **忌神**：土（印 — 生金克水）、金（比劫增战）

### 大运应期
- **丙戌运**：丙火七杀制金，戌土生金 — 少年平顺
- **丁亥运**：丁火正官、亥水食神通关 — 好转
- **戊子运**：戊土印星生金，子水通关 — 平稳
- **己丑运**：己丑土旺生金，金旺克木 — 金木再战，波折
- **庚寅运**：庚金比肩坐寅木财 — 财被分夺
- **辛卯运**：辛金劫财坐卯木财 — 财被分夺

### 命理启示
两气成象的核心在于 **通关**：
1. 金木相战 — 水可以通关
2. 水火相战 — 木可以通关
3. 火金相战 — 土可以通关
4. 土水相战 — 金可以通关
5. 木土相战 — 火可以通关

《滴天髓》云：**"关内有织女，关外有牛郎，此关若通也，相邀入洞房。"**
