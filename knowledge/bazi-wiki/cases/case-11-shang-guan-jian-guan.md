# 伤官见官—灾祸不断（命例11）

## 任铁樵原评（主旨）

癸水生于午月，财星当令；年干甲木伤官高透，时干戊土正官；月干庚金正印孤立无根。

任氏评曰：**伤官见官，为祸百端。此命甲木伤官遥克戊土正官，官星受创，一生官非口舌不断。虽有庚金印星，却被旺火所熔，自顾不暇，焉能生水？**

## 解读

### 格局分析
- 癸水生于午月 — 偏财当令，身弱
- 年干甲木伤官 — 高透有力
- 时干戊土正官 — 正官合身
- 月干庚金正印 — 孤立无根，虚浮

此为 **伤官格**。但伤官见官，格局大忌。

### 伤官见官的四种情况

| 情况 | 吉凶 | 本例 |
|------|------|------|
| 伤官旺、官星弱 | 官星受伤 → 官非口舌 | ✓ 甲木克戊土 |
| 官星旺、伤官弱 | 伤官被制 → 有惊无险 | ✗ |
| 有印通关 | 伤官生印、印生身 → 转吉 | ✗ 庚金无力 |
| 有财转化 | 伤官生财 → 财生官 | ✗ 财已旺 |

本例属于最凶的组合：**伤官有力正面攻击官星，又无印星通关化解。**

### 大运应期
- **辛未运**：辛金印星虚浮 — 少年多灾
- **壬申运**：壬水劫财帮身 — 稍有好转
- **癸酉运**：癸水比肩帮身 — 平稳但无大发展
- **甲戌运**：**甲木伤官得戌土根气** — 伤官更旺，与戊土正官对抗，官非不断
- **乙亥运**：乙木食神合庚，亥水帮身 — 晚年转运
