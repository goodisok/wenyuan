# 正财格富贵命（命例02）

## 八字

| | 年柱 | 月柱 | 日柱 | 时柱 |
|--|------|------|------|------|
| 天干 | 庚 | 乙 | 庚 | 丙 |
| 地支 | 申 | 酉 | 申 | 戌 |

**日主：** 庚金（阳金）
**五行分布：** 金极旺，木受克，火被耗

## 大运

| 大运 | 年龄 |
|------|------|
| 丙戌 | 0-9 |
| 丁亥 | 10-19 |
| 戊子 | 20-29 |
| 己丑 | 30-39 |
| 庚寅 | 40-49 |
| 辛卯 | 50-59 |

## 任铁樵原评（主旨）

庚金生于酉月，得令；年月日三见申酉，金旺之极。月干透乙木正财，时干透丙火七杀。此命之妙在于乙木正财为用，丙火七杀为辅，财生杀、杀护财，财杀两旺而身更强。

任氏评曰：**财旺生官（杀），官（杀）护财，富贵之命。**

## 解读

### 格局分析
- 日主庚金生于酉月（金旺之月），得令
- 年支申、日支申 — 金有强根
- 时支戌 — 土生金
- 月干乙木正财 — 财星有根（乙木在戌中有微根）
- 时干丙火七杀 — 杀星有根（丙火在戌中有根）

此为**财格配杀**，身强足以任财杀。

### 用神
- **喜神**：火（杀/官）、木（财）
- **忌神**：土（印）、金（比劫）

### 分析要点
1. 庚金旺极，喜财（木）来耗、官杀（火）来克
2. 乙木正财透出，丙火七杀高透 — 财杀有情
3. 财生杀，杀卫财 — 格局清纯
4. 大运走火木运 — 助财杀，富贵可期

### 大运应期
- **丙戌、丁亥运**：火运助杀，少年聪慧
- **戊子运**：子水伤官生财，渐入佳境
- **己丑运**：土旺生金，稍显平淡
- **庚寅、辛卯运**：木运助财，富贵巅峰
