# 从杀格—弃命从杀（命例15）

## 八字

| | 年柱 | 月柱 | 日柱 | 时柱 |
|--|------|------|------|------|
| 天干 | 癸 | 己 | 丁 | 辛 |
| 地支 | 巳 | 未 | 卯 | 亥 |

**日主：** 丁火（阴火）
**五行分布：** 火土旺、水木金各有根

## 大运

| 大运 | 年龄 |
|------|------|
| 戊午 | 7-16 |
| 丁巳 | 17-26 |
| 丙辰 | 27-36 |
| 乙卯 | 37-46 |
| 甲寅 | 47-56 |
| 癸丑 | 57-66 |

## 任铁樵原评（主旨）

丁火生于未月，食神当令；月干己土食神高透，年干癸水七杀遥制，时干辛金偏财生杀。

任氏评曰：**此命丁火有年支巳火、日支卯木生扶，看似不弱。然全局己未土旺极，泄火太过，癸水七杀遥攻，辛金生杀。日主虽得生扶，却被群土泄尽，终不得不从于杀势。**

此为 **假从杀格** — 日主有根但被泄尽，从杀之格。

## 解读

### 格局分析
- 丁火生于未月 — 食神当令
- 年支巳火 — 日主之根
- 日支卯木 — 印星生身
- 月干己土 — 食神泄身
- 年干癸水七杀 — 遥制
- 时干辛金偏财 — 生杀助势

**从杀条件：**
1. 杀旺 ✓（癸水七杀得辛金生）
2. 日主弱极 — 丁火虽有巳火卯木，但全局己未土泄火
3. 日主无援 — 援星被泄
4. 顺杀之势 — 不可逆杀

### 用神
- **喜神**：金（财生杀）、水（杀）
- **忌神**：木（印逆杀势）、火（比劫逆杀势）

### 大运应期
- **戊午运**：戊土伤官泄火 — 少年普通
- **丁巳运**：丁火比肩、巳火帮身 — 逆杀之势，多有不顺
- **丙辰运**：丙火劫财帮身 — 仍逆杀势
- **乙卯运**：**乙卯印星生身** — 逆杀大势，此运最凶
- **甲寅运**：甲寅印星更强 — 大凶
- **癸丑运**：癸水杀得丑土财星生 — 顺势从杀，晚年反而安稳

### 命理启示
此命说明 **从格的逆势之凶**：
- 从杀格的命，最怕运走印比（逆杀势）
- 从格的"顺"与"逆"决定了人生的吉凶走势
- 很多从格之人，早年不识格局而强行"奋斗"，反而处处碰壁
