# 调候命例—冬金寒木（命例08）

## 八字

| | 年柱 | 月柱 | 日柱 | 时柱 |
|--|------|------|------|------|
| 天干 | 丙 | 庚 | 辛 | 甲 |
| 地支 | 午 | 子 | 丑 | 午 |

**日主：** 辛金（阴金）
**五行分布：** 金旺、水旺、火有根、木有根、土有根

## 大运

| 大运 | 年龄 |
|------|------|
| 辛丑 | 4-13 |
| 壬寅 | 14-23 |
| 癸卯 | 24-33 |
| 甲辰 | 34-43 |
| 乙巳 | 44-53 |
| 丙午 | 54-63 |

## 任铁樵原评（主旨）

辛金生于子月，寒冬水冷金寒；年干丙火正官透出暖局，时干甲木正财生火。

任氏评曰：**子月辛金，天寒地冻，非丙火不能解冻。妙在年时两午火，丙火高透，官星得用，寒金得暖，富贵无疑。此调候之紧要也。**

## 解读

### 调候分析

此命例是**调候理论**的经典案例。

**调候原理：**
- 辛金生于子月（冬月），水旺金寒
- 冬金最需要丙火来解冻暖局
- 命局年干丙火透出，年支午火、时支午火为根
- 丙火既是正官（格局），又是调候用神

**调候与格局的统一：**
丙火在此命中有双重作用：
1. **调候功能**：暖金解冻
2. **格局功能**：正官格

当调候用神与格局用神为同一字时，命格最高。

### 用神
- **调候用神**：丙火（暖局解冻）— 第一要务
- **格局用神**：丙火（正官）、甲木（财生官）
- **忌神**：水（增寒）、金（比劫）

### 大运应期
- **辛丑运**：比肩帮身，丑土生金 — 少年尚可
- **壬寅运**：壬水伤官合丙（伤官见官）、寅木财运 — 青年波折
- **癸卯运**：癸水食神，卯木财星 — 渐入佳境
- **甲辰运**：甲木正财生丙火官星 — 事业有成
- **乙巳、丙午运**：火运全力助官 — 富贵巅峰

### 命理启示
此命例说明了调候的重要性：
1. **冬生必看丙火**：无论何日主，冬天出生首先要看有没有火调候
2. **一火双用**：丙火既是调候又是正官，统一则贵
3. **调候先于格局**：按《穷通宝鉴》之说，先看调候再看格局

详见 [[qiong-tong-bao-jian]] 了解完整的调候理论。
