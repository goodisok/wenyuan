# 比劫夺财—一生破败（命例14）

## 八字

| | 年柱 | 月柱 | 日柱 | 时柱 |
|--|------|------|------|------|
| 天干 | 乙 | 庚 | 乙 | 丙 |
| 地支 | 卯 | 辰 | 巳 | 戌 |

**日主：** 乙木（阴木）
**五行分布：** 木旺、土旺、火旺、金孤立

## 大运

| 大运 | 年龄 |
|------|------|
| 己卯 | 7-16 |
| 戊寅 | 17-26 |
| 丁丑 | 27-36 |
| 丙子 | 37-46 |
| 乙亥 | 47-56 |
| 甲戌 | 57-66 |

## 任铁樵原评（主旨）

乙木生于辰月，财星当令；年干乙木比肩帮身，时干丙火伤官泄秀。月干庚金正官合身。

任氏评曰：**比劫夺财之命。以乙木日主坐巳火伤官，年支卯木禄神帮身，比劫林立，而财星辰戌土被木克伐，虽有庚金正官，却被丙火伤官所制。一生财来财去，终成空。**

## 解读

### 格局分析
- 乙木生于辰月 — 财星当令
- 年柱乙卯 — 比肩坐禄，帮身有力
- 月干庚金正官 — 合格
- 时干丙火伤官 — 克庚金正官

此命的要害在于 **比劫夺财**：

```
年乙卯（比劫帮身）→ 木旺克土（财）→ 辰戌土财星受伤
月庚辰（正官合身）← 被时干丙火伤官所克
时丙戌（伤官泄秀）→ 泄身生财但财又被比劫所夺
```

**循环困局：** 伤官生财 → 比劫夺财 → 财来财去

### 用神
- **喜神**：水（印）、金（官杀制比劫）
- **忌神**：木（比劫）、火（伤官克官）

### 大运应期
- **己卯运**：己土财星被卯木克 — 少年家贫
- **戊寅运**：戊土财星被寅木劫财克 — 求财辛苦
- **丁丑运**：丁火食神、丑土财星 — 稍有积蓄
- **丙子运**：子水冲午火（无午），子水印星帮身 — 转机
- **乙亥运**：亥水印星生身、乙木比肩 — 有水润局，晚年尚可
