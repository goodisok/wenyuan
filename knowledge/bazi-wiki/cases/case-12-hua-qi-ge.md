# 化气格—甲己化土（命例12）

## 八字

| | 年柱 | 月柱 | 日柱 | 时柱 |
|--|------|------|------|------|
| 天干 | 丁 | 戊 | 甲 | 己 |
| 地支 | 未 | 申 | 戌 | 巳 |

**日主：** 甲木（阳木）
**五行分布：** 土极旺、火旺、金有根、水无、木无根

## 大运

| 大运 | 年龄 |
|------|------|
| 丁未 | 2-11 |
| 丙午 | 12-21 |
| 乙巳 | 22-31 |
| 甲辰 | 32-41 |
| 癸卯 | 42-51 |
| 壬寅 | 52-61 |

## 任铁樵原评（主旨）

甲木生于申月，七杀当令，然全局土旺极。年干丁火伤官、月干戊土偏财、时干己土正财。甲己贴身而合。

任氏评曰：**甲己合而化土，化气成格。全局火土成势，甲木无根，不得不从土而化。化气真者，富贵无疑。**

## 解读

### 格局分析

**化气格的条件：**
1. **天干相合**：甲己合
2. **化神当令**：土旺于四季末（戌月）或火土月
3. **化神有根**：地支戌未巳 — 火土成势
4. **被化者无根**：甲木在地支无寅卯亥子等根气
5. **不被破坏**：无庚辛金克甲木（不使甲木恋旧）

本例全部满足：
- 甲己合化土 ✓
- 月令申金生水（但不克土），时支巳火生土 ✓
- 戌未巳三土局 ✓
- 甲木在地支无寅卯根 ✓

### 用神
- **喜神**：火（印生土）、土（比劫）
- **忌神**：木（反化）、水（克火泄土）

### 大运应期
- **丁未运**：火土运助化神 — 少年顺遂
- **丙午运**：火土大运 — 家境殷实
- **乙巳运**：乙木合庚（不破甲己合），巳火助化 — 事业有成
- **甲辰运**：甲木透出争合己土 — 略有反复
- **癸卯运**：**癸水克丁火、卯木助甲木** — 破格之运，大凶
- **壬寅运**：壬水克丁火 — 晚年破败

### 命理启示
化气格的秘密在于"真化"与"假化"：
- **真化**：化神有力，被化者完全放弃自己 — 大贵
- **假化**：化神有力但有破绽 — 富贵而不长久
- **不化**：名合实不合 — 以常格论

此命为真化，所以早中年顺遂，但癸卯运破格则前功尽弃。
