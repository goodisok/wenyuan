# 杀印相生—武贵命（命例04）

## 八字

| | 年柱 | 月柱 | 日柱 | 时柱 |
|--|------|------|------|------|
| 天干 | 壬 | 丙 | 丙 | 乙 |
| 地支 | 午 | 午 | 寅 | 未 |

**日主：** 丙火（阳火）
**五行分布：** 火极旺、木旺、水弱、金无、土有根

## 大运

| 大运 | 年龄 |
|------|------|
| 丁未 | 6-15 |
| 戊申 | 16-25 |
| 己酉 | 26-35 |
| 庚戌 | 36-45 |
| 辛亥 | 46-55 |

## 任铁樵原评（主旨）

丙火生于午月，羊刃格；年干壬水七杀通根于微，时干乙木正印高透。

任氏评曰：**羊刃格七杀制刃，印星化杀，杀印相生，武贵之命。虽水火相战，然印星通关有力，化杀生身，文武双全。**

## 解读

### 格局分析
- 丙火生于午月 — 羊刃格（日主临帝旺）
- 年支午、月支午、日支寅 — 寅午半合火局
- 年干壬水七杀 — 克身制刃
- 时干乙木正印 — 化杀生身
- 时支未 — 火之余气

**杀印相生路径：** 壬水七杀 → 生乙木正印 → 生丙火日主

### 用神
- **喜神**：木（印）、水（杀）
- **忌神**：火（比劫夺杀）、金（财坏印）

### 分析要点
1. 丙火虽旺，有壬水七杀制衡
2. 乙木正印通关 — 杀不攻身，反生身
3. 杀印相生为最贵格局之一
4. 年柱七杀、月柱羊刃 — 威权在握

### 大运应期
- **戊申、己酉运**：食伤制杀，少年志气
- **庚戌运**：庚金偏财 — 财生杀，但戌为火库，尚可
- **辛亥运**：辛金合丙，亥水助杀 — 官杀两旺，权势巅峰
- **壬子运**：杀旺攻身 — 晚年退隐
