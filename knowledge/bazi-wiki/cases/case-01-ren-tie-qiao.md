# 任铁樵自评命造（命例01）

## 八字

任铁樵本人的八字，在《滴天髓阐微》中有自评段落。这是他一生经历的命理印证。

| | 年柱 | 月柱 | 日柱 | 时柱 |
|--|------|------|------|------|
| 天干 | 癸 | 丙 | 戊 | 丁 |
| 地支 | 未 | 辰 | 午 | 巳 |

**日主：** 戊土（阳土）
**五行分布：** 火土极旺，水木微弱

## 大运

| 大运 | 年龄 |
|------|------|
| 乙卯 | 5-14 |
| 甲寅 | 15-24 |
| 癸丑 | 25-34 |
| 壬子 | 35-44 |
| 辛亥 | 45-54 |
| 庚戌 | 55-64 |
| 己酉 | 65-74 |
| 戊申 | 75-84 |

## 任铁樵原评（主旨）

任氏自评此命：戊土日主生于辰月，得丙丁火生扶，巳午未火土成方局，日主极旺。全局火炎土燥，水木无根。此命之妙在于火土成势，从旺而不可逆。

任氏以此命说明：**旺极不可克，顺其势而为之。**

## 解读

### 格局分析
- 日主戊土生于辰月（土旺之月），得令
- 年支未、日支午、时支巳 — 巳午未会南方火局
- 天干丙丁火透出 — 印星有力
- 时干丁火正印贴身 — 印旺身强
- 年干癸水虚浮无根 — 财星极弱

此为**从旺格**（火土旺极），不可逆势。

### 用神
- **喜神**：火（印）、土（比劫）
- **忌神**：水（财）、木（官杀）— 逆火土之势

### 大运应期
- **乙卯、甲寅运**（5-24岁）：木运生火，尚可
- **癸丑运**（25-34岁）：丑为湿土晦火，癸水犯旺，多有坎坷
- **壬子运**（35-44岁）：水火相战，人生波折
- **辛亥运**（45-54岁）：辛金耗土，亥水冲巳，变动极大

任氏在此命理中阐述了一个核心理念：**命局过旺不可强行扶抑，须从其势而引导。** 这一思想对后世影响深远。
