# 八字命理知识库

基于 [Karpathy's LLM Wiki](https://gist.github.com/karpathy/442a6bf555914893e9891c11519de94f) 模式构建的八字命理学知识体系。

## 内容结构

```
wiki/bazi/
├── SCHEMA.md          # 领域约定与标签分类
├── index.md           # 内容目录
├── log.md             # 操作日志
├── raw/               # 原始资料（不可修改）
│   └── tools/         #   开源工具参考（bazi-skill）
├── concepts/          # 核心概念
│   ├── 基础篇         # 天干地支、五行、四柱、十神、大运流年
│   └── 古籍经典篇     # 渊海子平、滴天髓、子平真诠、三命通会等九本经典
├── entities/          # 实体参考
│   ├── 十二生肖
│   ├── 十二长生
│   └── 格局
├── methods/           # 方法论
│   └── 排盘步骤
└── cases/             # 实战命例（滴天髓阐微）
    └── 16个命例覆盖财格、印格、官杀格、从格、调候等全部主流格局
```

## 覆盖范围

- **5 大基础概念** — 天干地支、五行生克、四柱结构、十神体系、大运流年
- **9 本经典古籍** — 从宋代的渊海子平到民国的千里命稿，梳理每部经典的核心理论和历史贡献
- **16 个实战命例** — 源自《滴天髓阐微》（任铁樵注），包含八字排盘、任氏原评、白话解读、用神分析和大运应期

## 使用方式

### Obsidian（推荐）
用 Obsidian 打开 `wiki/bazi/` 目录：
- `[[wikilinks]]` 交叉引用
- Graph View 可视化概念网络
- 配合 Dataview 插件可做标签查询

### 命令行
```bash
# 搜索概念
grep -r "伤官" concepts/
# 搜索命例
grep -r "庚金" cases/
```

### 配合 Hermes Agent
```bash
# Agent 可自动维护和扩展知识库
cd ~/wiki/bazi
# 提交新内容
git add -A && git commit -m "..." && git push
```
